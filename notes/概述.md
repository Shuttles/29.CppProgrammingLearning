以下类的实现都是==泛型编程==写出来的，重点在于==把类型区别淡化==

queue类：队列

1. 头文件：queue

2. 命名空间：std

3. 声明：

   ```c++
   #include <queue>
   using std::queue;
   queue<data_type> q;
   ```

   

stack类：栈

与queue基本相同



string类：字符串

1. 头文件：string

2. 命名空间

3. 声明

   ```c++
   #include <string>
   using std::string;
   string s1, s2;
   
   s1 = "Hello world!\n"
       
   s1 == s2; //判等
   s1 < s2; //字典序小于
   s1 += s2; //字符串连接
   s1.length();//字符串长度 O(1),因为string是个类    而strlen是O(n)
   ```
