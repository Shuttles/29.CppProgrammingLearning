# 1.基础

1. 显式模板实例化

```c++
add<double>(2, 3)
```

2. `decltype()`

   自动推导表达式返回值类型

3. 模板是作用在编译时！跟宏差不多！

   但是作用在宏之后。

4. 在单一源文件中必须得用模板的定义！否则无法在编译时实例化出定义！

5. 所以模板的定义一般实现在头文件中！



如何看.o文件中的定义？

```c++
nm -C xxx.o
```
